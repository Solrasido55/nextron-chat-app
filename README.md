# Nextron + Firebase 를 사용하여 구현한 실시간 채팅앱 

### 기간
2월 1일 ~ 2월 8일

### 기능
- 회원가입, 로그인
- 1대1 채팅
- 그룹채팅
- 유저목록 확인
- 로그아웃

### 사용기술
- Nextron, React, Typescript, Firebase, Recoil, Emotion

### 실행방법
```
git clone https://github.com/Solrasido55/nextron-chat-app.git
cd nextron-chat-app
npm install
npm run build
```
nextron-chat-app/dist 디렉토리 내의 파일 실행
